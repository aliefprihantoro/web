# web
personal web.

## structure project
> this project use monorepo.
- ui : (for ui/component/storybook)
  - home
  - blog
- apps : (for web/api)
  - home
  - blog
- packages : (shared packages)
  - html : for hilight
  - configs : configuration ts,eslint,tailwind
- e2e :
  - home
  - blog

## install
```bash
git clone https://github.com/alifprihantoro/web
cd web
pnpm i --lockfile-only
pnpm dev
```

- build :
